# Exercise 4: Arrange-Act-Assert (AAA) Pattern, Test Fixtures, Setup and Teardown Methods in JUnit

## Problem Statement

Organize unit tests using the Arrange-Act-Assert (AAA) pattern and use setup and teardown methods with JUnit annotations.

## Expected Output

`	ext
Setup completed
Teardown completed

Test Passed Successfully
`

## Observation

- The AAA pattern improves test readability by separating test preparation, execution, and verification.
- The @Before method runs before each test and is used for setup activities.
- The @After method runs after each test and is used for cleanup activities.
- Test fixtures help create a consistent environment for executing test cases.
