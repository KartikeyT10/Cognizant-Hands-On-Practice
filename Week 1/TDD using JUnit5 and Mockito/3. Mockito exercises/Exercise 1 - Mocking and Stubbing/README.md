# Exercise 1: Mocking and Stubbing

## Problem Statement

Use Mockito to mock an external API, stub its methods with predefined values, and verify the behavior of a service class.

## Expected Result

`	ext
Test Passed Successfully
`

## Observation

- Mockito is used to create a mock implementation of the external API.
- Stubbing allows predefined responses to be returned without calling the actual API.
- This helps test application logic independently of external dependencies.
- Mocking improves test reliability and execution speed.
