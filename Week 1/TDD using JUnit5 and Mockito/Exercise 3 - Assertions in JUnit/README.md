# Exercise 3: Assertions in JUnit

## Problem Statement

Use different JUnit assertions to validate test results and verify expected behavior.

## Assertions Used

| Assertion | Purpose |
|------------|----------|
| assertEquals() | Checks whether two values are equal |
| assertTrue() | Checks whether a condition is true |
| assertFalse() | Checks whether a condition is false |
| assertNull() | Checks whether an object is null |
| assertNotNull() | Checks whether an object is not null |

## Expected Result

`	ext
All test cases passed successfully.
`

## Observation

JUnit assertions help verify the correctness of code by comparing actual results with expected results. Different assertion methods can be used to test various conditions and improve the reliability of applications.
