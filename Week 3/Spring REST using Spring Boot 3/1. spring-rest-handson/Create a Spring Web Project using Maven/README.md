# Hands-on 1: Create a Spring Web Project using Maven

## Problem Statement

Create a basic Spring Boot web application using Maven and understand the default project structure generated by Spring Initializr.

## Project Configuration

The project is created using Spring Initializr with the following details:

- Group: com.cognizant
- Artifact: spring-learn
- Build Tool: Maven
- Dependencies:
  - Spring Web
  - Spring Boot DevTools

## Expected Output

`	ext
Starting Spring Learn Application
Started SpringLearnApplication
Spring Learn Application started
`

## Observation

- Spring Initializr simplifies the creation of a Spring Boot project.
- Maven is used for dependency management and project building.
- src/main/java contains the application source code.
- src/main/resources contains application configuration files.
- src/test/java contains test classes.
- The @SpringBootApplication annotation is used to configure and start the Spring Boot application.
