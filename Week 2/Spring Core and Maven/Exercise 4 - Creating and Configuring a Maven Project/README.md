# Exercise 4: Creating and Configuring a Maven Project

## Problem Statement

Create a Maven project for a Library Management application and configure it with the required Spring dependencies and Maven Compiler Plugin.

## Expected Result

`	ext
BUILD SUCCESS
`

## Observation

- Maven manages project dependencies automatically through the pom.xml file.
- Spring Context, Spring AOP, and Spring WebMVC dependencies are added to the project.
- The Maven Compiler Plugin is configured to use Java 1.8.
- Maven simplifies project building, dependency management, and compilation.
