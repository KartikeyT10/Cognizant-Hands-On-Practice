# Cognizant Java FSE Hands-On Practice

Welcome to my repository for the **Cognizant Java Full Stack Engineer (FSE) Upskilling Program**. 

## About This Repository

This repository serves as a personal portfolio and tracking system for all the weekly hands-on exercises and mandatory problem statements I am completing as part of my Java FSE learning journey at Cognizant. 

## What I Will Be Doing Here

Throughout this program, my workflow in this repository will involve:

1. **Solving Weekly Exercises**: I will be writing code to solve the mandatory and additional hands-on problems provided in the `Digital-Nurture-JavaFSE` curriculum.
2. **Applying Java FSE Concepts**: The code uploaded here will reflect the practical application of concepts learned each week, demonstrating my understanding of Java, full-stack development, and related technologies.
3. **Organizing Solutions**: All code will be structured and organized in a week-wise format (e.g., Week 1, Week 2 folders) for easy navigation and review.
4. **Self-Evaluation & Tracking**: I will be continually evaluating my solutions based on the problem criteria and maintaining this repository to share my progress with my Point of Contact (POC) as required.
